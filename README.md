# BaiDuMapSelectDemo
Android汤姆猫小游戏


#### 主要功能
+ 戳 tom 的各个身体部位它会做出相应反馈，包括动作和声音。
+ 往右滑动屏幕的时候另外一只黄色的小狗跳出来搞怪，同时发出声音。
